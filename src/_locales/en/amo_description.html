Cast Kodi, as the name suggests, enables to cast to Kodi:
  • videos: direct links (avi, mkv, ...), YouTube, Vimeo and Dailymotion;
  • musics: direct links (mp3, flac, ...) and SoundCloud.
A new item "Cast to Kodi" is added on context menu of link / page / selection of videos or audio. Then two choices are available : Play or Queue Item. In Kodi, you must Allow remote control via HTTP. This option is on Settings / Services / Control. Then you must configure Cast Kodi Options by copying values filled in Kodi : Port and possibly Username and Password. You must provide IP Address of server hosting Kodi ; this data is displayed on Settings / System info / Summary page. To play videos / musics hosting on platforms, it needs plugins YouTube, Vimeo, DailyMotion.com or SoundCloud in Kodi.
